Polígonos de hidrografía

Si alguna de las siguientes geometrías tiene área, tendemos que crear un polígono de hidrografía mediante la opción Creación de polígonos de hidrografía:

Tipo de geometría
Lago, laguna
Zona pantanosa
Balsa, alberca
Embalse
Estanque
Fuente
Canal, acequia
Playa

Para crear un polígono de hidrografía seguiremos los siguientes pasos:

  1. Seleccionamos la opción Crear polígono de hidrografía.
    Esa opción realiza las siguientes tareas:
    • Activar únicamente la visualización de códigos de hidrografía y edificaciones.
    • Generar una topología por inundación.
    • Asignar como código activo el código Hidrografía_POL
    • Ejecutar la orden PONER_ATR_R
  2. Seleccionamos el polígono que queremos definir. Para ello, al pasar el cursor por el área que quieres formar como Polígono de hidrografía, ésta se debe rellenar. Puede ocurrir tres cosas:

    1. Que se inunde exactamente el área que quieres convertir en polígono de hidrografía. En este caso haz clic para marcar el relleno y pulsa la tecla espacio para confirmar.
    2. Que se inunde únicamente parte del área que quieres convertir en polígono de hidrografía. En este caso tendrás que hacer Clic en el primer área y Control + Clic cada una de las zonas que definen el polígono. Una vez tengas todas las zonas marcadas, pulsa la tecla espacio para confirmar.
    3. Que la zona que quieres inundar esté mal cerrada y al pasar el cursor sobre ella se inunde un área mucho más grande de lo que quieres inundar.

      Si al pasar el cursor por el área a inundar no se inunda nada o se inunda un área mucho mayor de lo que te interesa es posible que el área que te interesa no esté cerrada. Esto puede ser por lo siguiente:

      1. Porque no tengas todos los códigos necesarios encendidos.
        Este caso no debería suceder: En caso de que suceda, comunícalo a tu contacto técnico, pues en teoría los polígonos de hidrografía se deben definir perfectamente si tenemos activos los códigos de Hidrografía y de Edificaciones, que son los códigos que activa la opción Creación de polígonos de hidrografía.

        Para resolver este problema:
        1. Pulsa la tecla Esc para finalizar la orden de inundación.
        2. Enciende todos los códigos mediante la opción del menú Ver/Todos los códigos
        3. Selecciona la opción Crear polígono de hidrografía (sin apagar códigos) que como su nombre indica, no enciende ni apaga ningún código.
      2. Porque las líneas que delimitan el polígono que quieres crear no están cortadas entre sí.

        Para resolver este problema:
        1. Pulsa la tecla Esc para finalizar la orden de inundación.
        2. Ejecuta cualquiera de las órdenes de Digi3D.NET para partir la línea, como EXT_P.
    4. Pulsamos la tecla Barra espaciadora para confirmar que el polígono está bien definido
    5. Seleccionamos en el desplegable el tipo de polígono que acabamos de inundar.
    6. Aceptamos el cuadro de diálogo.
