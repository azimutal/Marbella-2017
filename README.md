# Plugins-Digi3D
Tabla, órdenes y control de calidad para el proyecto de Marbella.

* Configura Digi3D.NET para multi-codificación **Herramientas/Configuración/DigiNG/Interfaz de multi-codificación**.
* Configura Digi3D.NET para que dibuje el último código primero **Herramientas/Configuración/DigiNG/Orden dibujo (multicódigo)**
* Al abrir el archivo de dibujo asigna al campo **Tabla de Códigos** en la pestaña **Archivo de Dibujo** el siguiente valor: https://github.com/azimutal/Marbella-2017/raw/master/Marbella.dt.
* Una vez abierto el archivo de dibujo, activa el panel de **Códigos activos** en **Ventana/Códigos activos**.
* Activa el panel de **Tentativos** en **Ventana/Tentativos**.
* Activa el panel **Cuadro de herramientas* en **Ventana/Cuadro de herramientas**.

Descarga la ayuda de la siguiente dirección https://github.com/azimutal/Marbella-2017/raw/master/Marbella.chm. 
Si tu ordenador tiene Windows 10, es posible que al abrir la ayuda no se muestre el contenido. Para solucionarlo:
* Pulsa con el botón derecho del ratón en el archivo .chm.
* Selecciona la opción **Propiedades**.
* Selecciona la opción **Desbloquear**.
* Acepta el cuadro de diálogo.

Puedes descargar el archivo de dibujo del tutorial de https://github.com/azimutal/Marbella-2017/raw/master/ArchivoTutorial.bind
